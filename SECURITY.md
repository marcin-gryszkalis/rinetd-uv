# Security Policy

## Supported versions

We support vesion 2 (2.0.0 or later) of **rinetd-uv**. Note, that this project started from version 2.0.0 to avoid confusion with ([https://github.com/samhocevar/rinetd)[Original rinetd] versioning.

## Reporting a Vulnerability

If you believe you have found a security vulnrability in **rinetd-uv**, please report it via [GitHub Vulnerability Reporting](https://github.com/marcin-gryszkalis/rinetd-uv/security) or via email mg@fork.pl (PGP/GPG public key: http://fork.pl/gpg.txt).

Please provide as much information as you can about the vulnerability.
Provide details about your configuration and environment, if applicable.
It's recommended to apply the principles of [Coordinated vulnerability disclosure](https://en.wikipedia.org/wiki/Coordinated_vulnerability_disclosure) model.

We take all security bugs seriously. Thank you for improving the security of **rinetd-uv**.
We appreciate your efforts and responsible disclosure and will make every effort to acknowledge your contributions.

## Original rinetd

**Rinetd-uv** is total rewrite of ([https://github.com/samhocevar/rinetd)[Original rinetd] - although they share common roots and non-negligible part of source code.
If you find a bug in the shared area then we'll try to coordinate fix and disclosure.
